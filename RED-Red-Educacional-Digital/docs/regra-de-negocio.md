# Regras de Negócio - RED

### RNE-001 - Média para aprovação
O aluno será aprovado se a média final for maior ou igual a 6.

### RNE-002 - Prazo de tarefas
Tarefas não podem ser enviadas após o prazo.

### RNE-003 - Transparência das notas
Responsáveis têm acesso completo às notas dos alunos.

### RNE-004 - Comunicação restrita
Mensagens são limitadas às turmas e usuários autorizados.

### RNE-005 - Biblioteca digital
Apenas professores podem publicar materiais.

### RNE-006 - Privacidade de dados
Dados dos usuários são protegidos e não podem ser compartilhados.

### RNE-007 - Registro de alterações
Alterações em notas e tarefas devem ser registradas.

### RNE-008 - Bloqueio de login
Após múltiplas tentativas inválidas, o acesso é bloqueado.
